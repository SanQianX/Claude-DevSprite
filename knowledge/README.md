# Claude-DevSprite 知识库

本项目知识库由 Claude-DevSprite 自动生成和维护。

## 项目简介

Claude-DevSprite 是一个 Claude Code Skill，自动监听 Git Commit，通过 AI 分析代码变更，生成/更新结构化知识库文档，配套 Web Dashboard 支持文档间跳转和源码预览。
