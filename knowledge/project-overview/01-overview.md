# 项目概览

## 一句话描述

**自动将 Git 提交转化为可浏览、可跳转、团队共享的项目知识库。**

## 项目定位

Claude-DevSprite 是一个 Claude Code Skill 插件，它的核心功能是：

1. **自动监听** Git 仓库的 Commit 事件（零配置，三层检测策略自动降级）
2. **AI 分析** 代码变更内容（复用 Claude Code 模型配置）
3. **生成/更新** 结构化 Markdown 知识库文档
4. **可视化浏览** 通过 Web Dashboard（localhost:38888）
