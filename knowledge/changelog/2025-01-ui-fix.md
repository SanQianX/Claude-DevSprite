### 2. web/src/components/viewer/SourceViewer.vue

**变更类型**: 修改

**变更详情**:
- 为源码内容容器添加了 `ref="contentRef"` 以支持 DOM 引用
- 新增了 `highlightLine` prop，用于接收需要高亮的行号
- 引入 `onMounted` 和 `watch` 生命周期钩子
- 实现了 `scrollToHighlight` 函数：在组件挂载和高亮行变化时，自动将指定行滚动到可视区域并高亮显示
- 新增行数: 51， 删除行数: 3

**影响范围**: 源码查看器组件。新增了按行号高亮并滚动定位源码的功能。

**相关文件**:
- [SourceViewer.vue](/project/Claude-DevSprite/source?path=web/src/components/viewer/SourceViewer.vue)

### 3. web/src/views/DashboardView.vue

**变更类型**: 修改

**变更详情**:
- **筛选器绑定**: 为 `statusFilter` 和 `severityFilter` 两个 `<select>` 元素添加了 `v-model` 双向数据绑定，使其能正常控制筛选逻辑
- **定位按钮功能**: 为“定位”按钮添加了 `@click` 事件处理器。点击后，通过 `router.push` 跳转到对应的源文件查看页面，并附带 `line` 查询参数以指示行号
- 为定位按钮组件传入了必要的 `file` 和 `line` 属性
- 新增行数: 41， 删除行数: 12

**影响范围**: 仪表盘视图。修复了筛选器无法工作的问题，并实现了从审查项直接跳转到源码指定行的功能。

**相关文件**:
- [DashboardView.vue](/project/Claude-DevSprite/source?path=web/src/views/DashboardView.vue)

### 4. web/src/views/ProjectView.vue

**变更类型**: 修改

**变更详情**:
- 为搜索输入框添加了 `v-model="searchQuery"` 绑定，用于收集用户输入
- 为搜索输入框添加了 `@keydown.enter="handleSearch"` 事件，支持回车键触发搜索
- 新增 `searchQuery` ref 变量
- 新增 `handleSearch` 函数，该函数调用 `router.push` 跳转到搜索结果页面
- 新增行数: 9， 删除行数: 0

**影响范围**: 项目视图。完善了搜索框的功能，使其支持回车触发搜索。

**相关文件**:
- [ProjectView.vue](/project/Claude-DevSprite/source?path=web/src/views/ProjectView.vue)

### 5. web/src/views/SourceView.vue

**变更类型**: 修改

**变更详情**:
- 为 `<SourceViewer>` 组件新增传递 `:highlight-line="highlightLine"` 属性
- 新增 `highlightLine` 计算属性，用于从路由查询参数 `line` 中解析出需要高亮的行号
- 新增行数: 5， 删除行数: 1

**影响范围**: 源码查看视图。支持从路由参数接收行号，并将其传递给 `SourceViewer` 组件以实现高亮定位。

## 功能改进总结

本次修复主要解决了 Web Dashboard 中多个关键交互元素的失灵问题，显著提升了用户体验：

1.  **搜索功能完善**：项目搜索框现在支持回车触发，并能正确跳转至搜索结果页。
2.  **筛选功能修复**：仪表盘中的审查项状态和严重性筛选器恢复了正常筛选能力。
3.  **源码跳转增强**：仪表盘的“定位”按钮现在可以准确地跳转到源码文件的指定行。
4.  **源码高亮新增**：源码查看器新增了行高亮与自动滚动功能，与源码跳转功能形成闭环。
