项目概览

Claude-DevSprite 是一个 Claude Code Skill 插件，自动监听 Git 仓库的 Commit 事件，通过 AI 分析代码变更，实时生成结构化的 Markdown 知识库文档。

核心价值

围绕四个核心环节构成完整闭环：自动监听、AI 分析、生成更新、可视化浏览。

分析引擎

核心编排逻辑位于 source:src/analyzer/pipeline.ts:42 (Pipeline 类)，负责协调 Diff 收集、上下文构建、AI 调用等步骤。

AI 提供者实现位于 source:src/analyzer/aiProvider.ts，支持 SDK 直连和 CLI 子进程两种模式。

知识库管理

文档存储使用 source:src/knowledge/storageManager.ts，关系图管理使用 source:src/knowledge/relationEngine.ts。
