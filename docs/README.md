# Claude-DevSprite 开发文档

本目录存放 Claude-DevSprite 项目的开发文档和设计文档。

## 目录结构

```
docs/
├── README.md                          ← 本文档
├── development-status.md              ← 开发进度总览
├── designs/                           ← 功能设计文档
│   ├── 01-multi-agent-system.md       ← 多 Agent Team 协作系统设计
│   └── ...
└── guides/                            ← 开发指南
    └── ...
```

## 文档分类

| 类别 | 位置 | 说明 |
|------|------|------|
| 开发进度 | `docs/development-status.md` | 已完成功能、待开发功能、技术债务 |
| 功能设计 | `docs/designs/` | 新功能的架构设计和技术方案 |
| 开发指南 | `docs/guides/` | 开发环境搭建、编码规范等 |
| 知识库 | `knowledge/` | 系统自动生成的项目知识文档（功能产出，非开发文档） |
