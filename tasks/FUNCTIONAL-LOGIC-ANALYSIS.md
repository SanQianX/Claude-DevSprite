## 数据流排查

### 前端 → 后端数据流

```
┌─────────────────────────────────────────────────────────────────────────┐
│                            数据流追踪                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  用户操作                                                                │
│    ↓                                                                    │
│  Vue 组件事件处理                                                        │
│    ↓                                                                    │
│  Pinia Store Action                                                     │
│    ↓                                                                    │
│  API 客户端 (apiClient)                                                  │
│    ↓                                                                    │
│  HTTP 请求 / WebSocket 消息                                              │
│    ↓                                                                    │
│  Express 路由 / WebSocket Handler                                        │
│    ↓                                                                    │
│  业务逻辑处理                                                            │
│    ↓                                                                    │
│  SQLite 数据库 / 文件系统 / 外部 CLI                                      │
│    ↓                                                                    │
│  响应返回                                                                │
│    ↓                                                                    │
│  Store 状态更新                                                          │
│    ↓                                                                    │
│  DOM 更新                                                                │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 关键数据流路径

#### 路径 1: 聊天消息
```
ChatInput.vue → chatStore.sendMessage() → POST /api/chat/send
    ↓
wsHandler.ts → TeamExecutor → Claude CLI subprocess
    ↓
WebSocket 流式返回 → chatStore 接收 → ChatMessageList.vue 更新
```

#### 路径 2: 审查修复
```
ReviewCard.vue → approveReview() → PUT /api/reviews/:id/approve
    ↓
reviews.ts → AI Provider → 执行修复
    ↓
更新数据库 → 返回结果 → UI 更新
```

#### 路径 3: 任务同步
```
ChatPanel.vue (AI 创建任务) → POST /api/projects/:name/tasks
    ↓
tasks.ts → 插入数据库 → 返回任务
    ↓
TaskList.vue 刷新 → 显示新任务 → Dashboard 统计更新
```

### 同步场景

| 触发源 | 目标 | 同步内容 |
|--------|------|----------|
| 聊天创建任务 | TaskList | 新任务显示 |

### 同步机制

```typescript
// 1. Store 共享状态
const chatStore = useChatStore()
const taskStore = useTasksStore()

// 2. 事件总线 (EventBus)
emit('task-created', task)
on('task-created', refreshTasks)

// 3. WebSocket 推送
ws.on('task.assigned', handleTaskAssigned)

// 4. 轮询刷新
setInterval(fetchDashboardStats, 30000)
```

## 排查优先级总结

### 🔴 P0 - 必须排查 (核心功能)
1. **DevChat 消息收发** - 发送→回复→显示→持久化
2. **DevChat 会话切换** - 切换会话→历史保留→切换回来
3. **AI 审查扫描** - 触发扫描→结果显示→详情查看
4. **AI 审查修复** - 批准→执行→进度→结果
5. **任务创建同步** - 聊天/审查创建任务→列表同步

### 🟠 P1 - 重要排查
6. **任务状态管理** - 状态切换→筛选→统计同步
7. **项目分析** - 触发分析→进度→结果→日志
8. **项目添加** - 路径选择→验证→添加→列表更新
9. **配置管理** - 保存→测试→生效
10. **实时日志** - SSE 连接→实时更新→过滤

### 🟡 P2 - 常规排查
11. **知识库浏览** - 文档列表→查看→搜索
12. **文件树** - 展开→折叠→导航

14. **搜索功能** - 输入→结果→跳转
