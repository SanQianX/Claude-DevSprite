## 控件清单

| 控件 | 类型 | 功能 | 事件 | 选择器 |
|------|------|------|------|--------|
| 状态筛选 | select | 按状态筛选审查项 | @change | .filter-select:first-child |
| 严重性筛选 | select | 按严重性筛选审查项 | @change | .filter-select:nth-child(2) |
| 重置按钮 | button | 清除所有筛选条件 | @click | .filter-reset-btn |
| 审查项 | div | 显示单条审查信息 | @click | .review-item |
| 批准按钮 | button | 批准修复审查项 | @click.stop | .btn-approve |
| 忽略按钮 | button | 忽略审查项 | @click.stop | .btn-ignore |
| 讨论按钮 | button | 跳转到聊天讨论 | @click.stop | .btn-discuss |
| 定位按钮 | button | 跳转到源码位置 | @click.stop | .btn-locate |

## 数据流

```
用户选择筛选条件
    ↓
v-model 绑定 severityFilter
    ↓
computed filteredReviews 重新计算
    ↓
DOM 更新显示筛选后的列表
```

```
用户点击批准按钮
    ↓
@click.stop 阻止事件冒泡
    ↓
approveReview(id) 函数
    ↓
dashboardStore.approveReview()
    ↓
API 调用 PUT /api/reviews/:id/approve
    ↓
更新本地 review.status = 'approved'
    ↓
filteredReviews 自动重新计算
    ↓
DOM 更新
```
