# Bug Fix: 自动修复勾选和扫描状态刷新后丢失

## 问题描述

Dashboard 页面的两个状态在刷新页面后丢失：
1. **"自动修复"复选框** — 勾选后刷新页面，复选框恢复为未勾选
2. **扫描中状态** — 扫描进行中刷新页面，按钮从"扫描中..."恢复为"开始扫描"

## 影响范围

- 用户体验：刷新后需要重新勾选自动修复，且无法知道扫描是否仍在进行
- 数据完整性：自动修复的意图在页面刷新后丢失

## 修复状态

✅ 已修复并通过 Playwright UI 测试验证
