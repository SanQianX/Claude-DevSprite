## AppHeader 组件结构

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  [DevSprite Logo]    [🔍 搜索栏...]         [● Analyzing] [中] [🌙] [⚙]   │
└─────────────────────────────────────────────────────────────────────────────┘
```

| 控件 | 类型 | 功能 | 生命周期 |
|------|------|------|----------|
| `.logo` | router-link | 首页导航 | - |
| `SearchBar` | 组件 | 全局搜索 | - |
| `.analysis-indicator` | div | 分析进度指示 (SSE 驱动) | onMounted → connectSSE() |
| `.icon-button` (语言) | button | 中/英切换 | - |
| `.icon-button` (主题) | button | 明/暗切换 | - |
| `router-link` (设置) | router-link | /settings 页面 | - |

### SSE 连接生命周期 (修复前)

```
AppHeader.onMounted()
    ↓
analysisStore.connectSSE()  → 建立 SSE 连接
    ↓
用户导航到其他页面 (如 /project/xxx)
    ↓
AppHeader.onUnmounted()
    ↓
analysisStore.disconnectSSE()  ← 错误！断开了全局 SSE 连接
    ↓
新页面的 AppHeader.onMounted()
    ↓
analysisStore.connectSSE()  ← 重新建立连接 (增加了不必要的开销)
```

## CodePanel 组件结构

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  📁 源码                                                [✕]                │
├─────────────────────────────────────────────────────────────────────────────┤
│  ← 文件树    src/components/App.vue                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│  1 │ <template>                                                            │
│  2 │   <div class="app">                                                   │
│  3 │     <router-view />                                                   │
│  4 │   </div>                                                              │
│  5 │ </template>                                                           │
├─────────────────────────────────────────────────────────────────────────────┤
│  📄 关联文档 (3)                                                            │
│  ├─ router.ts — L12, L45                                                   │
│  ├─ main.ts — L8                                                           │
│  └─ App.spec.ts — L15                                                      │
└─────────────────────────────────────────────────────────────────────────────┘
```

| 控件 | 类型 | 功能 |
|------|------|------|
| `.file-tree` | 组件 | 文件树浏览器 |
| `.code-body` | div | 代码查看器 (行号 + 代码) |
| `.related-docs` | div | 关联文档列表 |
| `.back-btn` | button | 返回文件树 |
| `.related-item` | div | 打开关联文档 |

### 文件加载流程

```
用户点击文件树节点
    ↓
selectFile(node)
    ↓
fileRequestCounter++
    ↓
fetch(`/api/projects/:name/source-file?path=...`)
    ↓
检查 reqId === fileRequestCounter  ← 防止竞态
    ↓
更新 currentFile + codeLines + relatedDocs
```
